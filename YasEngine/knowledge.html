Swapchain

Knowledge source: Learning Vulkan by Parminder Singh GPUOpen

Swapchains - are mechanisms which i used during rendering to (drawing primitives - so exactly to present final image) to show on platform-specific window. Swapchain is platform dependant. SwapchainKHR is object which represent set of images that can be presented on the window.

Color depth

Knowledge source: Wikipedia EN

Color depth or colour depth (see spelling differences), also known as bit depth, is either the number of bits used to indicate the color of a single pixel, in a bitmapped image or video framebuffer, or the number of bits used for each color component of a single pixel.
